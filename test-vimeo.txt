测试 Vimeo 链接：
https://vimeo.com/117125857

这是一个测试文件，用于验证 Vimeo 视频链接的检测和提取功能。

功能特性：
- 自动检测 Vimeo 视频链接
- 提取视频信息（标题、作者、分类、发布日期）
- 获取视频缩略图
- 自动获取内嵌视频链接（player.vimeo.com）
- 支持插入图片块和视频块
- 创建 Vimeo 标签并设置属性

技术细节：
- 标签来源：Vimeo API 的 categories 字段
- 内嵌链接：优先使用 player_embed_url，回退到 oEmbed API
- 发布日期：优先使用 release_time，回退到 created_time
